# Hangman-Word_guessing_game-
To design and implement a Hangman game where the user guesses the name of an animal letter by letter. The program should track correct and incorrect guesses and display the result (win or lose).

This project is a simple Animal Hangman game. The program randomly selects an animal name from a predefined list. The user must guess the word one letter at a time.
For each guess:
• If the letter is correct, it is revealed in the word
• If the letter is incorrect, the number of attempts decreases
The player is given a limited number of attempts (6 chances).
The game continues until:
• The user guesses the complete word → User wins
• The user runs out of attempts → Game over
This project demonstrates key programming concepts such as loops, conditional logic, string handling, and user input validation.
